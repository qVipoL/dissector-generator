### Wireshark lua dissector generator

#### How to run:

* 1 - Compile (make)
* 2 - Run the compiled file (out/bin/diss-gen)
* 3 - First parameter is the input file (.proto), and the second is the output file (.lua). The second parameter is not mandatory
